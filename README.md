# SYFO Storebror
Eksponerer datagrunnlag for statistikk og analyse fra innkommende sykepengesøknader og sykmeldinger.

## Teknologi
* Kotlin
* Ktor
* Gradle
* Spek
* Kafka
* Postgreqsql
* Vault

## Tilgang til Postgresql
Se [Postgres i NAV](https://github.com/navikt/utvikling/blob/master/PostgreSQL.md).
